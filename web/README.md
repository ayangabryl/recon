# Vocorize-inspired landing

Built from `docs/research/vocorize/DESIGN.md` + agent-captured reference screenshots.

## Run

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) (or the port Next.js prints).

## Stack

- Next.js 16 + App Router
- Tailwind CSS v4
- Instrument Serif + Inter (Google Fonts)
- Spline hero embed (same scene URL as reference)
